Week 4A: Understanding Data Types

The Foundation of Statistical Analysis

"Not all data is created equal"

Week's Overview

Welcome to Week 4A! Understanding data types is the absolute foundation of statistical analysis. The way we analyze data depends entirely on what type of data we're working with. Get this right, and everything else falls into place. Get it wrong, and your analysis could be completely invalid!

Data Types: The Foundation

All data falls into two main categories: Quantitative (numerical) and Qualitative (categorical). This is the 80/20 rule of data analysis - master this distinction and you're 80% of the way there!

üìä Data Type Hierarchy

ALL DATA
üî¢ Quantitative (Numerical)

Data you can measure or count

Discrete - Countable values (whole numbers)

Example: Number of students, goals scored

Continuous - Measurable values (any value in a range)

Example: Height, temperature, time

üìù Qualitative (Categorical)

Data that describes qualities or characteristics

Nominal - Categories without order

Example: Colors, car brands, gender

Ordinal - Categories with meaningful order

Example: Ratings, education levels, rankings

Why This Matters

Different data types require different statistical methods:

  • Quantitative data: You can calculate means, standard deviations, perform regression
  • Qualitative data: You work with frequencies, modes, chi-square tests
  • Using the wrong method = invalid results! ‚ùå

Quantitative Data

Quantitative data involves numbers. You can perform mathematical operations on it. But there's a critical distinction...

Discrete Data

Can only take specific, separate values

Usually whole numbers. You count discrete data.

Examples:

  • Number of students in a class
  • Goals scored in a cricket match
  • Number of cars in a parking lot
  • Books on a shelf

Continuous Data

Can take any value within a range

Can have decimals. You measure continuous data.

Examples:

  • Height (170.5 cm, 170.55 cm...)
  • Weight (65.3 kg, 65.35 kg...)
  • Temperature (25.6¬∞C, 25.67¬∞C...)
  • Time duration (10.23 seconds...)

Quick Test

Question: Is "age" discrete or continuous?

Answer: It depends on how you measure it!

  • Discrete: If you measure in whole years (e.g., "I'm 25 years old")
  • Continuous: If you include fractions (e.g., "I'm 25.5 years old")

In practice, age is usually treated as discrete!

3

Qualitative Data

Qualitative data describes qualities, categories, or characteristics. You can't do arithmetic on it. But order matters...

Nominal Data

Categories WITHOUT inherent order

Just labels or names. No "greater than" or "less than".

Examples:

  • Eye color (brown, blue, green)
  • Car brands (Maruti, Tata, Hyundai)
  • Gender (male, female, other)
  • Types of cuisine (Indian, Chinese, Italian)

Ordinal Data

Categories WITH meaningful order

Clear ranking or sequence. But intervals might not be equal!

Examples:

  • Customer satisfaction (very dissatisfied ‚Üí very satisfied)
  • Education level (high school ‚Üí bachelor's ‚Üí master's ‚Üí PhD)
  • Rankings (1st, 2nd, 3rd place)
  • Ratings (1 star, 2 stars, ... 5 stars)

The Key Difference

Nominal: "Red" isn't better or worse than "blue". No order.

Ordinal: "Very satisfied" is definitely better than "dissatisfied". Clear order!

Practice Problems

Problem 1: Classify the Following

Categorize each data type as Quantitative or Qualitative, then further classify them:

  1. Weight of apples in kg
  2. Brand of smartphone
  3. Customer satisfaction level (Poor, Fair, Good, Excellent)
  4. Number of emails received daily
  5. Blood type (A, B, AB, O)
  1. Quantitative - Continuous (measured, can have decimals)
  2. Qualitative - Nominal (categories, no inherent order)
  3. Qualitative - Ordinal (categories with clear order)
  4. Quantitative - Discrete (counted, whole numbers)
  5. Qualitative - Nominal (categories, no natural order)

Problem 2: Real-World Scenario

You're analyzing data for a Mumbai cricket stadium:

  • Number of spectators
  • Ticket price
  • Seat category (General, Premium, VIP)
  • Match duration in hours
  • Team names

Classify each data type.

  • Number of spectators: Quantitative - Discrete
  • Ticket price: Quantitative - Continuous
  • Seat category: Qualitative - Ordinal (General < Premium < VIP)
  • Match duration: Quantitative - Continuous
  • Team names: Qualitative - Nominal
